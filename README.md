# TVAI Luxe Mail ✨

A luxury-inspired email & messaging interface built with **HTML + TailwindCSS + Vanilla JS + Supabase**.

This project is a **UI/UX showcase + open-source experimental template**, not a production-ready messaging system.

---

## 🌟 Overview

TVAI Luxe Mail is a premium-style communication interface featuring:

- Royal / luxury themed UI design
- Thread-based messaging system (like email)
- AI-powered message composer (mock)
- Supabase integration support
- Local fallback mode (offline demo)
- Three.js animated gold particle background

---

## 🚀 Features

### 🖤 Frontend
- Fully responsive luxury dashboard UI
- Glassmorphism design system
- Animated particle background (Three.js)
- RTL Arabic support + multilingual UI
- Smooth modal system (auth, AI composer, tier upgrade)

### 💬 Messaging System
- Conversations (threads)
- Messages inside threads
- Mock data fallback mode
- Basic search/filter system

### 🔐 Authentication
- Supabase Auth integration (optional)
- LocalStorage fallback session system
- Guest mode (demo access)

### 🤖 AI Composer (Demo)
- AI-style message generator (rule-based mock)
- Tone selection (royal / poetic / corporate)
- Instant message insertion into composer

---

## 🧱 Tech Stack

- HTML5
- TailwindCSS
- Vanilla JavaScript
- Supabase (optional backend)
- Three.js (background animation)
- Lucide Icons
